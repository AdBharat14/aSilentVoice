# Technology Stack:
  - Flutter
  - Firebase Firestore 
  - Ultralytics YOLOV8
  - Object Detection
  - Image Classification and detection
  - EasyOCR for OCR (Optical Character Recognition)
  - Google collab for training the models
  - CVAT for building train dataset
